# Scripts del proyecto

Esta carpeta contiene los scripts de apoyo del proyecto:

- `python/`: scripts Python de utilidad, detección de ArUco, controlador ESP32, servidor y más.
- `automation/`: secuencias y herramientas de automatización para pruebas.
- `hardware-testing/`: scripts de prueba de hardware y E2E.

Usa `scripts/python` para ejecutar los scripts Python y `scripts/automation` / `scripts/hardware-testing` para las pruebas de integración y hardware.
